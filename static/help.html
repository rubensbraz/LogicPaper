1 The Excel Schema

DocGenius requires a strict 3-row header format in your .xlsx file.

// Row 1: Variable Names (Must match template tags)
client_name | contract_val | sign_date
// Row 2: Formatter Protocol
string;upper | currency;USD | date;long
// Row 3+: The Data
John Doe | 1500.50 | 2024-01-01
Supported Protocols:
  • string;upper, string;title
  • currency;BRL, currency;USD, currency;EUR
  • date;long (e.g., 12th January), date;iso
  • mask;###.###.###-## (Any pattern using #)
  • image;5;3 (Width;Height in cm)

2 Templates & Logic

Use Jinja2 syntax inside your Word or PowerPoint documents. The system replaces these tags with formatted Excel data.

Basic Replacement:
{{ client_name }}
Logic (Word Only):
{% if contract_val > 10000 %}
  High Value Client
{% endif %}

3 Image Handling

To insert dynamic images:

  1. Set Excel Row 2 to image;W;H.
  2. Put the filename in the Excel cell (e.g., photo.png).
  3. Create a ZIP file containing photo.png.
  4. Upload the ZIP in the "Assets Library" slot.
