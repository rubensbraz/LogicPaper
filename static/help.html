07. Dynamic Images

Inject images directly into Word documents using the `InlineImage` engine.

Step 1: Protocol

Use the image strategy.
image;[Width];[Height]
Example: image;5;3
(Width=5cm, Height=3cm)
Use auto to keep aspect ratio.

Step 2: Data

Cell content must match the filename.
signature_john.png
Requirement: This file must exist inside the uploaded assets.zip file.
